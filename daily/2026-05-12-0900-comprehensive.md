## 2) 推論（inference）
1. 結構判讀
   - 早盤前的跨市場結構偏向 **風險資產仍強、但交易主軸回到供應鏈與政策摩擦**。美股主要指數延續高檔，台股前一日續強，代表風險偏好沒有先崩，但國際面已開始把政策與供應鏈約束重新定價。
2. 風險因子
   - 第一個風險不是需求消失，而是 **地緣政治 + 能源價格 + AI 供應鏈瓶頸** 疊加。若油價高位維持、川普訪中議題升溫、HBM/AI server 交付再被放大，台灣 AI 供應鏈容易從「接單樂觀」轉成「交付與估值壓縮並存」。
   - 第二個風險是 **開發者供應鏈安全**。TanStack 事件代表本週科技團隊可能先修安全、再談效率，對 AI coding / agent 工具鏈採用節奏有短線干擾。
3. 資金風格
   - 目前資金風格偏向 **續抱 AI / tech beta，但更挑可交付、可量產、供應不掉鏈的標的**。題材仍在，但市場容忍度已從「講故事」慢慢切到「能不能準時出貨」。
4. 使用者真正關心的核心問題
   - 今天真正要盯的不是「AI 還熱不熱」，而是：**台灣供應鏈能不能把 AI 訂單順利轉成營收，而不是卡在 HBM、封裝、交期與電源配套。** 另一條是 **agent infra 熱度仍在，但敘事已進入安全與治理階段**，不是只看 star 數就夠。
