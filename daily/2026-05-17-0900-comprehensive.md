## 2) 推論（inference）
1. 結構判讀

   - 科技主線沒有消失，但敘事明顯從「模型能力」切到「agent 能不能部署、能不能控成本、能不能安全上線」。這對基礎設施、runtime、workflow 工具是正向，對只講概念的題材比較不利。
2. 風險因子
   - 第一風險是中東外溢到油價與航運，若荷莫茲海峽再有實質限制，通膨預期會直接壓縮科技估值。
   - 第二風險是 AI 供應鏈仍未真正鬆開。今天只有 HBM shortage 有新命中，NVDA / TSM / AVGO / MU / MRVL 於 5/15 普遍轉弱，說明市場對高估值 AI 硬體鏈的耐心正在下降。
   - 第三風險是 agent 生態的安全與政策雙壓。OpenClaw 一邊看到使用政策修正，一邊又看到公開 agent server 暴露風險，代表「會用」已不夠，還要「能治理」。
3. 資金風格
   - 資金暫時偏向防守與汰弱留強。AAPL、PLTR 相對抗跌，但高 Beta 的 AI 半導體鏈賣壓更重；這比較像資金先降低擁擠交易曝險，而非全面撤出科技。
   - 若後續油價續升、美債殖利率維持高位，短線資金更可能往現金流穩定、軟體平台或防禦型大型股移動，而不是追逐最熱的硬體供應鏈彈性。
4. 使用者真正關心的核心問題
   - 今天真正要盯的不是「AI 題材還紅不紅」，而是：AI 供應鏈是否再次進入估值修正、以及 agent / workflow 類標的能不能接棒成為下一段資金停泊點。
