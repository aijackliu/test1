## 1) 事實（fact）
- 市場資料
  - 目前可直接驗證的即時數字不足，未成功抓到台股 / 美股 / 匯率 / 利率 / BTC / Gold 的同步報價。
  - 來源限制：Reuters 頁面受 JS 擋截，web_search 亦遇到搜尋 API 缺 key，故今日市場數字缺口仍在。
- 事件面
  - TrendForce 於 2026-04-08 發布 AI 伺服器與高階 GPU 供應鏈更新，指出 NVIDIA 2026 年高階 GPU 出貨結構調整：Blackwell 比重由 61% 上修至 71%，Hopper+Rubin 合計比重下修。
  - 同文指出 Rubin 受 HBM4 validation、CX8 轉 CX9、功耗與液冷優化等因素影響，出貨時程有延後風險。
  - 文中亦提到地緣政治因素使 H200 時程存在不確定性。
- 科技熱點
  - AI server / high-end GPU 供應鏈仍是主軸，焦點在 Blackwell 擴產、Rubin 延遲風險、HBM4 驗證與新一代互連 / 散熱配置。
  - 固定追蹤關鍵字：HBM shortage, GPU lead time, AI server delay 有相關訊號，但 CoWoS delay 今日未取得直接可驗證新證據。
  - 今日未見新增高可信訊號（尤其是 CoWoS delay 的獨立更新）。

## 2) 推論（inference）
1. 結構判讀
   - 供應鏈主軸仍是 AI 算力擴張，但節奏從「純需求拉升」轉向「供應鏈瓶頸與代際切換管理」。
   - Blackwell 占比上升，代表市場資金與產能更偏向已成熟、可量產的路線，而非完全押注下一代未定案平台。
2. 風險因子
   - Rubin 延遲、HBM4 驗證、互連切換與液冷功耗，都是會把交期拉長的實體風險。
   - 若美中政策再度擾動，H200 與相關 AI 伺服器出貨可見度會下降。
3. 資金風格
   - 風格偏向「確定性優先」，資金可能先留在有訂單、可交付、可量產的龍頭與其供應鏈，而不是追最前沿但時程不明的題材。
4. 使用者真正關心的核心問題
   - 核心不是新聞本身，而是：AI 供應鏈這一波還能不能延續、延續的主力會落在哪些環節、以及延遲風險會不會壓縮估值。

2. 警戒點
   - 持續盯 Rubin 延遲是否擴大、HBM4 驗證進度、以及是否出現 CoWoS / AI server lead time 的新高可信訊號。
