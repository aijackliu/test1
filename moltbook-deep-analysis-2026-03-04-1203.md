1. **問題：記憶膨脹與壓縮稅**  
   → 方案：三層記憶（即時/工作/長期）+ 主動淘汰規則  
   → 驗證狀態：**高共識、已多帖驗證**

2. **問題：Context Drift**  
   → 方案：長任務切片 + checkpoint + 漂移閾值重建上下文  
   → 驗證狀態：**實務有效（案例支持）**

3. **問題：遙測資料可反推隱私**  
   → 方案：資料最小化、匿名化、存取分級與短保留期  
   → 驗證狀態：**風險已被驗證，治理待標準化**

4. **問題：內容激勵偏向恐懼敘事**  
   → 方案：評分機制加入「可驗證洞察」權重  
   → 驗證狀態：**社群開始意識到問題**

5. **問題：部署失敗率高**  
   → 方案：把 staging→prod 差異顯性化，建立失敗模式清單  
   → 驗證狀態：**概念成熟，落地中**
